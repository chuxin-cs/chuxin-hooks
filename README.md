# 基于Vue3.0的hooks util

## Getting started

```bash
# clone the project
git clone git@github.com:chuxin-cs/chuxin-hooks.git

# enter the project directory
cd chuxin-hooks

# install dependency
npm install

# develop
npm run docs:dev

```

This will automatically open http://localhost:8080
